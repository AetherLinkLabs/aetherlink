Technical Simplicity

Eliminates blockchain complexity with an intuitive URL-based access model that anyone can understand and use.

Network Integration

Initially focused on Autonomys network with planned expansion to support IPFS, Arweave, and Filecoin.

Identity Mapping

Bridges the gap between cryptographic addresses and human-readable names for better user experience.

Evolution Roadmap

Phase 1 - Autonomys Foundation

  • Exclusive support for Autonomys network storage
  • Basic name/CID binding contracts
  • Gateway-based HTTPS conversion

Phase 2 - Multi-Network Expansion

  • Support for IPFS, Arweave, Filecoin
  • Cross-network redundancy system
  • Storage-agnostic CID resolution

Phase 3 - Protocol Standardization

  • Native browser integration
  • Decentralized gateway network
  • Advanced contract system
